Habit Reset for iPhone

Cara pakai tercepat di iPhone:
1. Simpan folder ini ke iCloud Drive atau upload ke GitHub Pages / Netlify.
2. Buka file index.html lewat Safari.
3. Tekan Share, lalu Add to Home Screen.
4. App akan terasa seperti aplikasi biasa di iPhone.

Catatan penting:
- Data disimpan lokal di Safari browser perangkat.
- Kalau hapus data browser, data app juga bisa ikut hilang.
- Gunakan tombol Export secara berkala untuk backup.

Kalau kamu mau versi native iOS, saya bisa buatkan template SwiftUI juga.
